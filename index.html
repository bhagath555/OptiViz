What is Optimization

Optimization is the process of making something as effective or functional as possible. In mathematics and computer science, it refers to the selection of the best element from a set of alternatives. This can involve minimizing or maximizing a function by systematically choosing input values from within an allowed set and computing the value of the function.

In the context of numerical optimization, we often deal with functions that are continuous and differentiable. The goal is to find the input values (or parameters) that minimize (or maximize) the output of the function. This is crucial in various fields such as machine learning, engineering design, and economics.

Optimization Method Categories

Optimization algorithms can be broadly categorized into three main types based on their approach to finding the minimum or maximum of a function:

First-Order (Gradient) Methods
  • Steepest Descent (1D/2D)
  • Conjugate Gradient (1D/2D)
Second-Order (Hessian) Methods
  • Newton method (1D/2D)
  • Levenberg-Marquardt Method (1D/2D)
Zeroth-Order (derivative-free) Methods
  • Golden cut (1D)
  • Powell (2D)
  • Nelder-Mead (2D)

Inputs

Each optimization method requires specific inputs to function effectively. Problem dimension (1D/2D), objective function, initial guess are common inputs for all the algorithms. Other inputs vary by the method. Below are few dropdown options for the inputs.

1D functions
  • Parabola : (\( 2 + x^2 \) )
  • Wavy bowl : (\( 0.05\,x^2 - \cos(x) \))
2D function
  • Sphere : \( x^2 + y^2 \)
  • Rosenbrock : \( (1 - x)^2 + 100\,(y - x^2)^2 \)
  • Himmelblau : \( (x^2 + y - 11)^2 + (x + y^2 - 7)^2 \)
  • SinCos bowl : \( \sin(x)\cos(y) + 0.1\,(x^2 + y^2) \)
Line search method
  • Constant : requires step length
  • Armijo
  • Exact line search
Beta method (Conjugate gradient)
  • Fletcher–Reeves
  • Polak–Ribière
  • Hestenes–Stiefel
  • Dai-Yuan
