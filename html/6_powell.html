Powell Method

Powell’s Method is a derivative-free optimization algorithm designed to minimize a multivariable function without computing gradients. It applies a sequence of one-dimensional minimizations (golden cut) along a set of parameter directions, iteratively refining the solution.

2. Method

The algorithm proceeds by iteratively minimizing the function along a set of **conjugate directions**. At each iteration, Powell:

  • Performs a sequence of 1D line minimizations along each direction
  • Updates the directions based on the change in position
  • Searches along a newly constructed direction that captures combined progress

3. Algorithm

The Powell method can be summarized in the following steps:

  1. Initialize the starting point and direction vectors.
  2. For each iteration:
    • Perform a line search along each direction vector to find the minimum.
    • Update the direction vectors based on the new position.
    • Construct a new direction vector that combines previous directions.
  3. Repeat until convergence criteria are met.

4. Discussions

  • Does not require gradient or Hessian information
  • Effective for smooth low-dimensional problems
  • Capable of superlinear convergence under certain conditions

  • Can be inefficient in high-dimensional or non-smooth problems
  • Sensitive to initial direction set
  • May converge to local minima if the function is not unimodal in direction searches

5. Applications

Powell’s method is commonly used in:

  • Engineering design optimization
  • Curve and surface fitting
  • Machine learning hyperparameter tuning (when gradients are unavailable)
